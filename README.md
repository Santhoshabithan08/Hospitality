# Hospitality Revenue Analysis

## Project Description
This project analyzes hospitality business data to evaluate revenue performance, occupancy trends, and booking patterns. The analysis helps hotel management improve pricing strategies and operational efficiency.

## Tools Used
- Excel (Data Cleaning & Validation)
- SQL (MySQL) – Data Analysis
- Power BI – Dashboard & Visualization

## Key KPIs
- Total Revenue
- Occupancy Rate
- Average Daily Rate (ADR)
- Revenue per Available Room (RevPAR)
- Booking Trend by Channel

## Key Insights
- Identified high-performing properties and room categories
- Highlighted periods of low occupancy impacting revenue
- Analyzed booking channels contributing maximum revenue
- Provided insights to improve room utilization and pricing

- Dataset
- SQL Queries
