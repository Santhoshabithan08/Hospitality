# Hospitality Analytics 

## Project Summary

Analyzed hotel booking and revenue data to evaluate occupancy performance, customer behavior, and revenue trends. This project focuses on identifying key business insights to improve operational efficiency and customer satisfaction in the hospitality industry.

## My Role
- Performed end-to-end data analysis using Excel, Power BI, Tableau, and MySQL
- Built interactive dashboards to track key performance indicators
- Transformed raw data into meaningful insights for business decision-making
  
## Key Work Done
- Developed dashboards in Excel, Power BI, and Tableau to visualize booking trends and performance metrics
- Used MySQL to query and analyze data from multiple tables (bookings, hotels, rooms, dates)
- Analyzed occupancy rate, revenue generated vs realized, and booking patterns
- Evaluated customer ratings and booking status (Cancelled, Checked-Out, No Show)
- Compared weekday vs weekend booking behavior

## Key Insights
- Identified revenue loss due to cancellations and no-shows
- Observed higher booking trends during weekends
- Found that room categories and service quality impact customer ratings
- Highlighted opportunities to improve occupancy and revenue performance

## Tools Used
- Excel 
- Power BI 
- Tableau 
- MySQL 
- PowerPoint

## Outcome
This project enhanced my ability to work with real-world datasets, build dashboards, and generate data-driven insights to support business decisions in the hospitality domain.
